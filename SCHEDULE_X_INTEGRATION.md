# Schedule-X 行事曆整合說明

## 🎯 整合概覽

已成功將 Schedule-X 開源行事曆模塊整合到智慧社區管理系統中，創建了一個現代化的行事曆界面。

## 📋 功能特色

### 🗓️ 兩區塊佈局

**第一個區塊 - 月曆表**
- 使用 Schedule-X 的月網格視圖
- 顯示當前月份的所有行事曆事件
- 支持點擊事件查看詳情
- 響應式設計，支持手機/桌面

**第二個區塊 - 數據列表**
- **月行事曆模式**: 顯示即將到來的5筆資料
- **過期行事曆模式**: 顯示所有過期事件
- 獨立滾動條，高度固定400px
- 自定義滾動條樣式

### 🔄 分頁功能

1. **月行事曆**: 
   - 第一區塊顯示 Schedule-X 月曆
   - 第二區塊顯示即將到來的事件（最多5筆）

2. **過期行事曆**: 
   - 第一區塊顯示提示信息
   - 第二區塊顯示所有過期事件列表

### 🎨 視覺設計

- ✅ 與現有主題系統完全兼容
- ✅ 支持黑暗/明亮模式切換
- ✅ 使用方正粗體字體
- ✅ 響應式設計（手機/平板/桌面）
- ✅ 事件狀態顏色編碼
- ✅ 懸停效果和過渡動畫

### 📱 響應式斷點

- **桌面 (>768px)**: 完整兩區塊佈局
- **平板 (≤768px)**: 緊湊佈局，高度調整
- **手機 (≤480px)**: 垂直優先，最小化高度

## 🗂️ 文件結構

```
src/views/Frontstage/
└── ScheduleXCalendar.tsx      # 主要的 Schedule-X 行事曆組件

src/assets/styles/
└── schedule-x-custom.css       # 自定義樣式文件
```

## 🚀 訪問方式

1. **本地開發**: http://localhost:5175/calendar
2. **功能切換**: 使用頂部的「月行事曆」/「過期行事曆」按鈕

## 📊 數據流

- 事件數據存儲在 React state 中
- 支持事件的 CRUD 操作（新增、編輯、刪除）
- 自動區分當前/過期事件
- 實時更新日曆視圖

## 🎯 技術實現

- **React 18** with TypeScript
- **Schedule-X** 核心日曆引擎
- **CSS Variables** 主題系統
- **Tailwind CSS** 響應式設計
- **Custom Hooks** 狀態管理
